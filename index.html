Coding camp – фронтенд разработка сайтов

Описание курса

Добро пожаловать на курс фронтенд разработки сайтов, где вы узнаете, как создавать современные и красивые веб-сайты. Мы вместе создадим 3 мега-полноценных адаптивных веб-сайта с использованием трех основных технологий: HTML, CSS и JS. Научим детей основным технологиям создания веб-асайтов и дадим им возможность создавать свои собсвенные сайты, Эти знания позволят им заинтересоваться в современных веб-технологиях. Мы создадим с нуля 3 различных полноценных веб-сайта, а не только их части. Они будут полны современных и красивых эффектов. Мы начнем с относительно простых технологий, а также рассмотрим некоторые сложные части.

После того как дети овладеют основными технологиями верстки сайтов. Мы начнем изучение серверной части. Научимся создавать современные, масштабируемые и высокоскоростные веб-приложения с помощью Angular и Node.js + Express + MongoDB. Мы объединим все разработанные сайты с мощью серверной части NodeJS + Express + MongoDB

Настроим приложение NodeJS + Express + MongoDB + Angular с помощью Angular CLI Эффективно используя NodeJS и Express. C помощью инструментов, предоставляемых Angular подключим серверную часть NodeJS а также подключим базу данных MongoDB.

Чему научатся?

  • HTML5, CSS3 и современный JavaScript на 2021 год.
  • Создавайть адаптивные, доступные и красивые сайты.
  • Управляйть событиями на странице с помощью JavaScript
  • Использовать Express и MongoDB для работы на сервере.
  • Думать и развиваться дальше как разработчик
